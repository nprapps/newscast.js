Newscast.js

About

A library to radically simplify Chromecast web app development.

See a simple example. (You'll need a Chromecast, of course.) See a more complex example with bi-directional communcation. Read the API documentation.

Development tasks

Grunt configuration is included for running common development tasks.

Javascript can be linted with jshint:

grunt jshint

Uniminified source can be regenerated with:

grunt concat

Minified source can be regenerated with:

grunt uglify

API documentation can be generated with:

grunt jsdoc

The release process is documented on the wiki.
